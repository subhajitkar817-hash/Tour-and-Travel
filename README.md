Developed a full-stack web application for managing travel packages and Transport bookings. Users can browse packages, view available Transportation, select seats, and confirm bookings.
Implemented frontend using HTML, CSS, JSP for dynamic UI, backend using Java Servlets for business logic, and MySQL for database management.
Features include user authentication, session management, real-time seat availability tracking, and automated booking confirmation.
Ensured responsive design for better user experience and secure handling of user data.
